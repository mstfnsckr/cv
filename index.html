Hakkımda

Merhaba ben Mustafa, Kırıkkale Üniversitesi'de Bilgisayar Mühendisliği lisans eğitimimi tamamladım. Yazılım alanında kendimi geliştirmek ve güncel teknolojileri takip etmek için ilgi duyduğum diller ve teknolojilerle çeşitli projeler geliştirmekteyim. Eğitim sürecimde makine öğrenmesi, yapay zeka, derin öğrenme ve görüntü işleme alanlarında seçmeli dersler aldım. Ayrıca okul projeleri, staj döneminde kurum için geliştirdiğim yazılımlar ve kişisel projelerimde Python kullanarak yapay zeka alanında çeşitli eğitimler ve uygulamalar gerçekleştirdim. .NET ekosisteminde kendimi geliştirmeye odaklandım ve ASP.NET Core MVC, Flutter, MSSQL, Docker gibi teknolojiler üzerinde çalışmalar yaptım. Bunun yanı sıra web scraping, asenkron yapılar gibi konularda da projeler geliştirirken eğitimler aldım. Hem Windows hem de iOS işletim sistemleri üzerinde çalışmaya özen gösterdim. Mobil, web ve masaüstü uygulamaları geliştirirken teknik yetkinliğimi artırmaya devam ediyorum. Ayrıca Java, C# ve Python derslerini tamamladım; HTML, CSS ve JavaScript alanlarında da bilgi edinerek kendimi sürekli geliştirmeye odaklandım.
